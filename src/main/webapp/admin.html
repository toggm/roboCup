Админка

Зарегистрировать игрока

Команда работает только в фазе регистрации, т.е. тогда же когда работает и обычный !register в общаке заграма.
НЕ использовать по личным просьбам! Это только для тестов или в случае каких-то сбоев.

Создать игру на заграме

Создать игру на заграме с двумя выбранными игроками. Настройки партии идентичны турнирным. Важно: никаких новых записей в турнирной сетке не произойдёт. Если созданную игру доиграть до конца, то её результат попадёт в турнирную сетку. Если ваша цель -- это просто внести запись в турнирную сетку, то можете "назначить победителя" без создания партии (см.ниже). Введите имена игроков в виде Вася/Фрося.

Назначить победителя

Введите имена игроков в виде "победитель>проигравший". Например, Вася Новиков>Фрося Марусевна. Игрок "Вася Новиков" будет назначен выигравшим во встрече. На заграме, при этом, эта игра закончена не будет.
Осторожно!

Сообщение будет отправлено в публичный чат заграма.
